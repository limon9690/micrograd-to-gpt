# Neural Networks: Zero to Hero - Worked Implementations

Building neural networks from scratch - from a scalar-valued backprop engine up to a working GPT - following Andrej Karpathy's Neural Networks: Zero to Hero. Each folder is one video: the worked exercise notebook plus my own notes on what tripped me up and what I explored beyond the lecture.

Course and original exercises by Andrej Karpathy (karpathy/nn-zero-to-hero). This repository contains my own worked solutions, notes, and extensions.
